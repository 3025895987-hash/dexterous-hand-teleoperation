# 项目说明（可直接复用到作品集 / 报名表）

## 项目名称
基于手部姿态估计的灵巧手遥操作系统

## 项目简介
这是一个结合视觉识别与嵌入式控制的主从式灵巧手项目。系统通过摄像头捕捉人手动作，利用 MediaPipe 提取手部关键点，并计算手指关节角度；随后通过串口把角度发送给下位机，由下位机控制舵机驱动机械手，实现对人手动作的同步复现。

## 技术栈
- Python
- OpenCV
- MediaPipe
- PySerial
- MicroPython
- PWM 舵机控制

## 系统流程
1. 摄像头读取实时画面；
2. MediaPipe 识别手部关键点；
3. 计算手指关节角度；
4. 上位机通过串口发送角度报文；
5. 下位机解析报文并驱动舵机；
6. 机械手完成同步动作。

- 编写下位机 MicroPython 固件；
- 完成舵机控制与系统联调；
- 将项目整理为适合 GitHub 展示和继续迭代的形式。

## 项目亮点
- 实现了“视觉感知 -> 角度映射 -> 机械执行”的闭环；
- 具备较强的演示性和人机交互直观性；
- 适合作为机器人、嵌入式、计算机视觉交叉方向的入门项目；
- 可进一步扩展为五指版灵巧手或更复杂的抓取控制系统。
